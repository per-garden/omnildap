== README

Omnildap is an LDAP front-end to one or several LDAP or non-LDAP backends. Please refer to wiki pages and doc directory for further information.

* Omnildap currently uses ruby 2.1.7p400 (2015-08-18 revision 51632) [x86_64-linux].

* Rails 4.1.5

* Current implementation uses MySQL for data persistence

* Basic Rspec tests exist for both Web and LDAP frontends.

* Omnildap listens as a regular web server (port 3000) as well as a simple LDAP server (port 1389)
